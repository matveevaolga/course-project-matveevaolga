# NFR Traceability Matrix

| User Story | Связанные NFR | Приоритет | Веха |
|------------|----------------|-----------|-------|
| Как пользователь, я хочу голосовать за фичи, чтобы влиять на развитие продукта | P1, P2, S1, R1 | Высокий | MVP |
| Как пользователь, я хочу видеть статистику голосов, чтобы анализировать популярность идей | P3, R3 | Средний | v1.1 |
| Как администратор, я хочу защитить систему от накруток, чтобы обеспечить честность голосования | S1, S2 | Высокий | Security |
| Как пользователь, я хочу работать с системой в любое время, чтобы не прерывать процесс голосования | R1, R2 | Высокий | Reliability |
| Как разработчик, я хочу масштабировать систему, чтобы поддерживать рост пользователей | SC1, SC2 | Средний | Scalability |

## Легенда приоритетов:
- **Высокий:** Критично для MVP и безопасности
- **Средний:** Важно для пользовательского опыта
- **Низкий:** Улучшения и дополнительные возможности
