# Risk Register and Security Controls

| Угроза | Приоритет | Мера/Контроль | Связь с NFR из P03 |
|--------|-----------|----------------|-------------------|
| Подделка пользователя | High | Аутентификация пользователей | S1: Защита от накрутки |
| Повторное голосование | High | Валидация уникальности голосов | S1: Защита от накрутки |
| Потеря данных | Medium | Персистентное хранилище (БД) | R3: Целостность данных |
| DoS атаки | Medium | Rate limiting | P2: Обработка нагрузки |
| Вредоносный контент | Medium | Валидация и санитизация ввода | S2: Валидация данных |
| Манипуляция статистикой | Low | Аудит и логирование | R3: Целостность данных |
| XSS атаки | High | Экранирование вывода | S2: Валидация данных |

## Quick Wins (быстрые выигрыши):
1. **Rate limiting** - защита от DoS (2-4 часа)
2. **Валидация ввода** - базовые проверки (1-2 часа)
3. **Логирование** - аудит операций (1 час)

## Долгосрочные меры:
- Аутентификация пользователей
- Миграция на персистентную БД
- Шифрование чувствительных данных
